Problema 13

Energía cinética de un automóvil

Datos del problema

\(m = 750\,kg\)
\(v = 65\,km/h\)

Conversión de velocidad

\[ v = 65\frac{km}{h} \] \[ v = 65\left(\frac{1000}{3600}\right) \] \[ v = 18.06\,m/s \]

Energía cinética

\[ K = \frac{1}{2}mv^2 \] \[ K = \frac{1}{2}(750)(18.06)^2 \] \[ K \approx 1.22\times10^5\,J \]

Diagrama físico

+x → velocidad = 65 km/h _________ / /| / AUTO / | /_______/ | | | | | | / |_______| / 🚗 Energía cinética del vehículo K = ½mv² m = 750 kg v = 65 km/h

Factor de cambio de energía

\[ K \propto v^2 \] Si: \[ v_2 = \frac{v}{2} \] entonces: \[ K_2 = \left(\frac{1}{2}\right)^2K \] \[ K_2 = \frac{1}{4}K \]
La energía cinética se reduce **a una cuarta parte**.

Velocidad para tener la mitad de energía

\[ K \propto v^2 \] \[ \frac{K_2}{K_1}=\frac{v_2^2}{v_1^2} \] \[ \frac{1}{2}=\frac{v_2^2}{v_1^2} \] \[ v_2=\frac{v_1}{\sqrt2} \] \[ v_2=\frac{65}{1.414} \] \[ v_2\approx46\,km/h \]

Resultados finales

Energía cinética: \[ K \approx 1.22\times10^5\,J \] Si la velocidad se reduce a la mitad: \[ K=\frac{1}{4}K \] Velocidad para tener la mitad de energía: \[ v\approx46\,km/h \]
